# Projekt "Projektowanie aplikacji w Oracle Application Express"
Wraz z jedną osobą wykonaliśmy aplikację bazodanową za pomocą OracleApex. 
Sama aplikacja nie jest już dostępna, ponieważ na okres jedynie jednego semestru prowadzący udostępnił nam licencje do tej platformy, która wygasła wraz z końcem semestru zimowego 24/25, ale pozostały na moim komputerze pliki, których używałem w sprawozdaniu. 



## Opis ze strony przedmioty: 
>Cele projektu
>Celem projektu zaliczeniowego części zajęć laboratoryjnych z przedmiotu Zarządzanie Bazami Danych SQL i NoSQL jest wykonanie prostej aplikacji, umożliwiającej definiowanie, przeglądanie i edytowanie danych, >przechowywanych w bazie danych, przez użytkownika naiwnego (nie znającego technologii bazodanowych). Przed zaimplementowaniem aplikacji konieczne będzie zaprojektowanie i zaimplementowanie schematu >relacyjnego, z którym będzie współpracowała aplikacja.

>Projekt ma być realizowany w grupach 2-osobowych. W uzasadnionych przypadkach liczba osób w grupie projektowej może ulec zmianie, jednak pociąga to za sobą zmianę wymagań co do stopnia złożoności projektu
>(np. inną minimalną liczbę encji).

>Tematykę (modelowaną rzeczywistość) projektu grupa wymyśla samodzielnie. Ma być ona na tyle pojemna, aby spełnić wymagania, zawarte w punktach dot. wymagań dotyczących schematu relacyjnego.
