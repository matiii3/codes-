# Project "Designing an Application in Oracle Application Express"

Together with one other person, we developed a database application using Oracle APEX.  
The application itself is no longer available, as the instructor provided us with licenses for this platform only for the duration of one semester. These licenses expired at the end of the winter semester 2024/2025.  
However, I still have the files on my computer that I used in the project report.

## Description from the course page:
> **Project objectives**  
> The goal of the project, which is part of the laboratory component of the *SQL and NoSQL Database Management* course, is to create a simple application that allows a naive user (i.e., one without database technology knowledge) to define, browse, and edit data stored in a database.  
> Before implementing the application, it is necessary to design and implement the relational schema that the application will use.  
>
> The project is to be carried out in 2-person teams. In justified cases, the number of people in a project group may be changed; however, this will also involve changes in the project complexity requirements (e.g., a different minimum number of entities).  
>
> The topic (the modeled domain) of the project is chosen by the group itself. It should be broad enough to meet the requirements related to the relational schema structure.
