Zostały one zrealizowane przeze mnie w latach 2023-2025. 

Wszystkie napisałem w 100% sam,z wyjątkiem dwóch - Aplikacja bazodanowa w SQL oraz Projekt Systemu Regułowego w Pythonie były efektem pracy zespołu dwuosobowego (ja + jeszcze jeden student).

Nie jest to 100% projektów czy programów, które stworzyłem, a jedynie prezentacja materiałów, które mają mniej więcej zarysować z jakimi obszarami sobie radzę oraz projekty, do których wciąż mam dostęp (duża część moich materiałów jest umieszczana, na platformie eKursy Politechniki Poznańskiej, która niestety jest kasowana zwykle z końcem semestru).
